# 数学模型
$\begin{aligned}input \quad x \in R^n\\hidden\quad W \in R^{M*N},b\in R^m\\output\quad w\in R^m, b \in R\end{aligned}$
For hidden layer:
$h = \sigma(W_1x+b_1)$
$o = W_2^Th+b_2$
如果是分类问题：
$y = softmax(o)$
($\sigma$ 是激活函数)
## 为什么激活函数需要非线性
将$\sigma$ 带入原式子得倒$o = w_2^TW_1x + b$
依旧线性，依然是一个线性模型
