# 问题
大量的矩阵乘法导致梯度爆炸或者梯度消失
## 梯度爆炸
### 情况
使用ReLU作为激活函数
### 问题
#### 1.值超出值域
尤其是16位浮点数
#### 2.对学习率敏感
学习率大 -> 大梯度
学习率小 -> 训练无进展
需要在训练过程中不断调整学习率
## 梯度消失
### 情况
使用sigmoid作为激活函数
### 问题
#### 1.梯度变为0
#### 2.训练没有进展
#### 3.顶层训练较好，对底层影响尤为严重
## 处理方法
### 1.将乘法变加法，ResNet，LSTM
### 2.归一化，梯度归一化，梯度裁剪
### 3.合理的权重初始和激活函数
将每层的输出和梯度都看作随机变量
让他们的均值和方差都保持一致
