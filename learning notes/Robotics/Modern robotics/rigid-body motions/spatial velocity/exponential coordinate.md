
This a method to represent [[angular velocity]] . We do not need to define that rotation axis coincides with $z$ axis. To represent an angular velocity, we can use a rotation axis called $\hat{\omega}_s$ and rate of rotation $\dot{\theta}$. We then times then and $\vec{\omega_s} = \hat{\omega}_s \dot{\theta}$ is the angular velocity versus the frame $\{s\}$. And we can find the linear velocity of the three axis $xyz$. 
$\vec{\dot{x_b}} = \vec{\omega_s} \times \vec{x_b}$
$\vec{\dot{y_b}} = \vec{\omega_s} \times \vec{y_b}$
$\vec{\dot{z_b}} = \vec{\omega_s} \times \vec{z_b}$
# change the frame of [[angular velocity]] 
For an angular velocity $\vec{\omega_s} = \hat{\omega}_s \dot{\theta}$, if we want to fund $\vec{\omega}_b$, weed to transform the [[space frame]] to $\{b\}$, $\vec{\omega}_b = R_{bs}\vec{\omega}_s$
