# describe the position
we can describe position of the body frame $p$ as combinations of position of space frame $x_s$ and $y_s$
$p = p_xx_s + p_yy_s$
The expression above can be written as:
$p = \begin{bmatrix} p_x \\ p_y \end{bmatrix}$
# describe the orientation
$x_b = cos\theta x_s + sin\theta y_s$
$y_b = -sin\theta x_s + cos\theta y_s$
The expression above can be write as:
$P = [x_b \quad y_b] = \begin{bmatrix} cos\theta & -sin\theta \\ sin\theta & cos\theta \end{bmatrix}$
