# definition 
For a matrix A, if it has $CA = I = AC$
we call $C =A^{-1}$ as the inverse matrix of $A$

## invertibility (whether inverse matrix exists)
if A is invertible, then $Ax = 0$ can only have the zero solution $x = A^{-1}0 = 0$ 
## Properties
$(ABC)^{-1} =C^{-1}B^{-1}A^{-1}$

## calculation of inverse matrix

 $\begin {bmatrix} A & I \end{bmatrix}$  to  $\begin {bmatrix} I & A^{-1} \end{bmatrix}$
