# definition
#### 结构
antecedent + conclusion
P IMPLIES Q or Q => Q 
#### 真假性
true if 
(1) p is false or 
(2) q is true

#### 证明方法1
1. 假设P
2. 推出Q
#### 证明放法2
证明他的逆反命题（contrapositive）
NOT(Q) IMPLIES NOT(P)
#### 证明方法3（proof by contradiction)
假如命题是假的，那么相应的虚假事实为真；既然虚假事实不能为真，所以命题为真
