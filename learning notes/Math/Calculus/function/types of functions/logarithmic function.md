### number e
for $f(1)^{'} = 1$ ,we express it as limit 
$f(1)^{'} = lim_{x \to 0} \frac{ln(1+x) - ln(1)}{x} = lim_{x \to 0} ln(1+x)^{\frac{1}{x}} = 1$
therefore
$e = lim_{x \to 0}e^{ln(1+x)^{\frac{1}{x}}} = lim_{x \to 0}(1 + x)^{\frac{1}{x}} = lim_{n \to \infty}(1 + n)^n$
